# Sprint 2 Retrospective 
#### Amy Nguyen, Bianca Rossetti, Axel Brochu, Marin Melentii

## What went well
- Everyone worked effectively on their issues
- There was clear communication all throughout, whether in person or online.
- The team debugged issues together very efficiently in order to prevent work blockage. 

## What did not go well
- Merge conflicts, we often had too many merge requests up that weren't getting merged in time, so that when something merges, all the rest gets merge conflicts.


## Things to change in next sprint 
- Merge merge requests faster, set merge requests to auto-merge to help prevent this issue.
